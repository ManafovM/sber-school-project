# Справочник контрагентов на Spring Boot и React
## Запуск приложения
Для запуска приложения нужно создать .env файл в директории src/main/resources. В данной директории находится пример .env файла.
В Intellij IDEA запускаем CounterpartiesApp или вводим в терминал **gradle bootRun**.
В терминале переходим к директории src/main/reactapp и вводим **npm install**, а затем **npm start**.
Приложение будет запущено на (http://localhost:3000/).
